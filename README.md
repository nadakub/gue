# Cautare intrebari licenta

Aplicatie statica HTML/CSS/JavaScript pentru cautarea intrebarilor de licenta.

## Functionalitati

- cautare live intr-un input box;
- text fara diacritice pentru cautare mai usoara;
- filtrare dupa categorie;
- filtrare dupa raspuns corect A/B/C/D/E;
- selectarea unei intrebari si afisarea tuturor detaliilor: categorie, numar, variante si raspuns corect;
- functioneaza local si pe GitHub Pages.

## Cum se publica pe GitHub Pages

1. Creeaza un repository nou pe GitHub.
2. Incarca fisierele `index.html`, `style.css`, `app.js` si `data.js`.
3. Mergi la **Settings > Pages**.
4. La **Build and deployment**, alege **Deploy from a branch**.
5. Selecteaza branch-ul `main` si folderul `/root`.
6. Salveaza. Linkul aplicatiei va aparea in pagina GitHub Pages.

## Rulare locala

Deschide direct `index.html` in browser.
